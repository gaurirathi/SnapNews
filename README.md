# SnapNews 📰

SnapNews is a MERN stack news aggregation web application designed to deliver concise news summaries with a clean and responsive user interface.

Users can browse the latest news articles and click on articles to read the full story from the original source.

---

## 🚀 Features

* 📰 News card layout 
* 📱 Fully responsive UI for desktop and mobile
* 🌐 Redirect to original news source
* ⚡ Fast and lightweight frontend built with React
* 🗄️ MongoDB integration for storing news data

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Material UI
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas
* Mongoose

---


## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/gaurirathi/SnapNews.git
cd SnapNews
```

---

### 2. Install dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

### 3. Configure Environment Variables

Create a `.env` file inside the `server` folder:

```env
DB_USERNAME=your_mongodb_username
DB_PASSWORD=your_mongodb_password
```

Make sure:

* MongoDB cluster is active
* Your IP is whitelisted in MongoDB Atlas

---

### 4. Run Backend

```bash
cd server
npm start
```

Server runs on:

```bash
http://localhost:8000
```

---

### 5. Run Frontend

Open another terminal:

```bash
cd client
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```
